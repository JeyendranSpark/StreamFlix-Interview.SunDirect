# StreamFlix - Android Streaming App Clone

An Android application demonstrating a multi-type carousel UI and video playback capabilities using MVVM architecture and Media3 SDK.

## Features
- **Dynamic Home Screen:** 4 distinct UI Carousel types (Hero, Poster, Landscape, Circle).
- **Video Player:** Custom Media3 (ExoPlayer) implementation.
- **Orientation Support:** Seamless transition between Portrait and Landscape playback.
- **Architecture:** MVVM with Repository pattern.
- **UI:** ConstraintLayout & Nested RecyclerViews.

## Tech Stack
- Kotlin
- Jetpack (ViewModel, Activity-KTX, ViewBinding)
- Media3 (ExoPlayer)
- Glide (Image Loading)
- Gson

## Setup
1. Clone the repository.
2. Open in Android Studio Iguana or later.
3. Sync Gradle.
4. Run on Emulator/Device (Internet connection required for video streaming).
